# Zufallsgenerator für Gruppen

**Erstellen Sie einen Zufallsgenerator für Gruppen in Ihrer Klasse.**

**Benutzeroberfläche:**

**•    Dropdown-Menü: Anzahl der Schüler (2-35)**

**•    Schaltfläche „Liste erstellen” (generiert Schüler 1, Schüler 2...)**

**•    LISTE der Schüler auf dem Bildschirm ANZEIGEN (Schüler 1, Schüler 2...)**

**\- Anzahl der Gruppen: 2-8**

**\- Schaltfläche „Gruppen erstellen“**

**Aktivieren Sie die folgende Option  „Tipp“: Geben Sie die gleiche Zahl in das rosa Feld für diejenigen Schüler ein, die nicht zusammen sein sollen.** 

**Ergebnis:**

**\- Gruppen werden in Karten angezeigt**

**\- Jede Gruppe hat eine andere Farbe**

**\- Schaltfläche „Exportieren” (kopieren oder herunterladen)**

**\- Schaltfläche „Neu mischen”**

**Optionen:**

**\- Bestimmte Personen aus einer Gruppe ausschließen**

**Helles, modernes, spielerisches Design.**
